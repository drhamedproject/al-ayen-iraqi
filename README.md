# مؤسسة جامعة العين العراقية الانسانية

## نبذة عن المشروع

موقع إلكتروني لمؤسسة جامعة العين العراقية الانسانية يحتوي على روابط التواصل الاجتماعي وآخر الأخبار.

## التقنيات المستخدمة

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## التثبيت والتشغيل

```sh
# تثبيت المكتبات
npm install

# تشغيل السيرفر المحلي
npm run dev

# بناء المشروع
npm run build
```
